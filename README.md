This project explores how our urban environment can shape health outcomes. Specifically, this project looks at how the walkability of three Texas cities - Dallas, Austin, and Houston - can influence the mental health of residents. 
Methods included coding in R to create visualizations highlighting the relationship between walkabliity and indicators of depression in these cities. 
The Shiny app for this project is located here: https://malashiny.shinyapps.io/test-1/
